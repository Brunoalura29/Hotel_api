# Hotel Management API

Este repositório contém o backend do desafio técnico para desenvolvimento de uma API de gestão de hóspedes, reservas, check-in e checkout.

O projeto será desenvolvido ao longo dos próximos dias, e receberá atualizações contínuas.

Tecnologias:
- Java 17
- Spring Boot
- PostgreSQL
- Maven

Funcionalidades previstas:
- Cadastro e busca de hóspedes
- Criação de reservas
- Check-in / Checkout
- Cálculo automático de diárias e taxas
